# API RESTful - Gestión de Pacientes

## Tecnologías
- PHP
- PDO
- MySQL
- JWT
- Bootstrap (Frontend)

## Funcionalidades
- Login con JWT
- CRUD de pacientes
- Validación de datos
- Protección contra SQL Injection
- Roles (admin / user)

## Usuario Administrador
Email: admin@sistema.com
Password: 1234567890

## Configuración de Base de Datos

1. Crear una base de datos llamada:
   prueba_tecnica

2. Importar el archivo:
   database/prueba_tecnica.sql

3. Configurar credenciales en:
   config/Database.php

## Cómo ejecutar

1. Clonar repositorio
2. Configurar base de datos
3. Ejecutar en XAMPP
4. Acceder a:
http://localhost/Prueba_tecnica/
